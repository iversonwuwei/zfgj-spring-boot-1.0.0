履约保证金

元

（0.8-2倍月租金）

1、银行审核履约保障金到账之后进行冻结，平台将履约保证金到账信息回执房屋租赁当事人，并作为三方进行监管。
2、履约保证金用作乙方履行本合同约定义务的担保，如乙方违反合同约定，甲方可从履约保证金中扣除应由乙方承担的租金、违约金、赔偿金和相关费用。合同期满，如出现履约保证金不足以抵扣前述款项及费用时，乙方需在10日内补足，否则甲方可将乙方违约情况上传平台，乙方信用记录将受影响。
3、本合同解除时，甲、乙双方在验房之后，协商一致共同签署《房屋租赁履约保证金返还协议》，并通过平台同时解冻账户资金，将约定金额转入乙方账户。
4、若超过合同期限，仅有一方向平台申请履约保证金解冻，且违约方在接到平台通知后仍未回应，则履约保证金在10日后，自行解冻，转入守约方账户。
5、如果甲、乙双方有任意一方未按合同约定提前解约，若申请人提前一个月通知另一方并在平台备案，则按照正常程序解约，履约保证金按上述程序解冻；若申请人未提前一个月告知另一方并备案，则履约保证金解冻后存入另一方账户。
