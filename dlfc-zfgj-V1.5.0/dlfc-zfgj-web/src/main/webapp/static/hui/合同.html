五、履约保证金

1．乙方签署本合同须支付人民币（大写）_______________________圆(￥____________元)，无息存入___________________________账户,作为履约保证金（金额应在0.8-2倍月租金之间）。银行审核履约保障金到账之后进行冻结，平台将履约保证金到账信息回执房屋租赁当事人，并作为第三方进行监管。

2．若履约保证金3日内没有到账，则本合同作废，房屋租赁当事人应将合同退回中介公司。

3．履约保证金用作乙方履行本合同约定义务的担保，如乙方违反合同约定，甲方可从履约保证金中扣除应由乙方承担的租金、违约金、赔偿金和相关费用。合同期满，如出现履约保证金不足以抵扣前述款项及费用时，乙方需在10日内补足。

4．本合同解除时，甲、乙双方在验房之后，协商一致共同签署《房屋租赁履约保证金返还协议》，并通过平台或中介公司经纪人协助同时解冻账户资金，将约定金额转入各方账户。

5．若超过合同期限，仅有一方向平台申请履约保证金解冻，则平台将短信提醒、快递邮寄、电话联系另一方，任何一种方式联系均视为通知送达。若另一方在10日之内仍不回应，则平台在申请解冻流程发起的10日后，将履约保证金解冻，转入申请人账户。

6．如果甲、乙双方有任意一方未按合同约定提前解约，若申请人提前一个月通知另一方并在平台备案，则按照正常程序解约，履约保证金按上述程序解冻；若申请人未提前一个月告知另一方并备案，则履约保证金解冻后存入另一方账户。
